## Getting Started

### Installation

1. Clone the repository to your local machine.
   ```bash
   git clone https://github.com/your-username/snapgram.git
   ```

2. Navigate to the project directory.
   ```bash
   cd snapgram
   ```

3. Install dependencies.
   ```bash
   npm install
   ```
